About

Good Games Database is a website designed by Dani Bottiger and Alia Babinet for CS 257 Software Design Fall 2021 at Carleton College.
It is a video games database to see rankings, comparisons, recommendations, and the general information about the game. Each page with user input has instructions on how to use it.
